# Proyecto ICC

## Descripción
Proyecto final del curso de introducción a ciencias de la computación.

## ¿Cómo ejecutar el programa?
Para ejecutar el usuario se debe de ejecutar los siguientes comandos:
```console
make
java -jar Main.java
```
